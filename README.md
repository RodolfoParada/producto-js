# 🛒 Sistema de Gestión de Órdenes (JS CRUD)

Este es un proyecto educativo desarrollado en **JavaScript Vanila**, diseñado para gestionar un flujo de ventas mediante el uso de Programación Orientada a Objetos (POO). El sistema permite crear productos, agruparlos en órdenes y gestionar el historial de ventas.

## 🚀 Funcionalidades (CRUD)

- **Create (Crear):** Agregar productos con nombre y precio a una orden activa.
- **Read (Leer):** Visualización dinámica de las órdenes generadas en tarjetas interactivas.
- **Update (Actualizar):** Cálculo automático de totales y límites de productos por orden (máx. 5).
- **Delete (Borrar):** Eliminación de órdenes específicas del historial.

## 🛠️ Tecnologías Utilizadas

* **JavaScript (ES6+):** Uso de clases, métodos estáticos, getters/setters y manipulación del DOM.
* **HTML5:** Estructura semántica para la interfaz de usuario.
* **CSS3:** Diseño responsivo basado en tarjetas (Cards) con efectos de transición.

## 📋 Estructura de Clases

El sistema se basa en dos clases principales:

1.  **Producto:** Define la entidad del artículo con ID autoincremental, nombre y precio.
2.  **Orden:** Gestiona un arreglo de productos, calcula el total y genera el template HTML para la visualización.
